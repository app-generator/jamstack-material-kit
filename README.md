## Build from sources
---

```bash
$ # clone the sources
$ git clone https://github.com/app-generator/jamstack-material-kit.git
$ cd jamstack-material-kit
$
$ # install modules
$ yarn
$
$ # start the app in development mode
$ yarn start
$
$ # build the app for production
$ yarn build
$ # app is runing on http://localhost:8080/ 
```
